Projects

Temperature Conversion

A very simple web-based application in design, created using HTML, CSS, Bootstrap, JavaScript, and JQuery to give the application a useful functionality to the user. This application allows the conversion between Celsius and Fahrenheit; It is built in such a flexible manner that another conversion unit such as Kelvin can be added with ease.

Currency Conversion

This project focuses on the use of PHP and a publicly available API (Yahoo! Finance API and Yahoo! Query Language (YQL)) which allow the use and creation of a currency conversion web-based application. This "app" currently only converts between USD, EURO and Colombian Peso (COP). The flexibility of using a web application allows for the ease of adding many more currencies that are more convenient to the users.

To-Do List

A web-based application that allows users to create a to-do list. Users can add, delete and mark items to the list as completed. The creation of this application was mainly focus on the use JavaScript and jQuery however other web-components such as HTML5, CSS, Bootstrap were also implemented.
